# Customer Sales Analysis Using SQL

## Overview
This project uses SQL queries to analyze business data and extract insights related to sales performance, pricing, and customer-related information.

## Skills Used
- SQL (SELECT, WHERE, ORDER BY)
- Data analysis
- Data filtering and aggregation
- Business logic implementation

## Key Insights
- Identified high-performing channels based on commission rates
- Calculated discounted pricing for improved business understanding
- Filtered and analyzed recent and active records using date conditions

## File Included
- analysis_queries.sql: SQL queries used for data analysis and insights
